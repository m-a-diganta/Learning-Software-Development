Let's work with variables and filters

Text filters for the line "{{line}}"

  • Uppercase: {{ line|upper }}
  • Lowercase: {{ line|lower }}
  • Capitalize: {{ line|capfirst }}
  • Length: {{ line|length }}
  • Title: {{ line|title}}
  • Word Count: {{ line|wordcount }}
  • Cut 'j': {{ line|cut:"j" }}
  • Truncate Words (4): {{ line|truncatewords:4 }}
  • Truncate Chars (6): {{ line|truncatechars:6 }}

Number filters for the number {{number}}

  • Floatformat (2): {{ number|floatformat:2 }}
  • Add 100: {{ number|add:100 }}
  • Divisible by 3: {{ number|divisibleby:3 }}

List filters for the list {{list}}

  • Length: {{ list|length }}
  • First Element: {{ list|first }}
  • Last Element: {{ list|last }}
  • Join Elements: {{ list|join:", " }}
  • Slice (first 2): {{ list|slice:":2" }}
