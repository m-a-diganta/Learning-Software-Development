Let's work with variables and filters

Text filters for the line "{{line}}"

  • Uppercase: {{ line|upper }}
  • Lowercase: {{ line|lower }}
  • Capitalize: {{ line|capfirst }}
  • Length: {{ line|length }}
  • Title: {{ line|title}}
  • Word Count: {{ line|wordcount }}
  • Cut 'j': {{ line|cut:"j" }}
  • Truncate Words (4): {{ line|truncatewords:4 }}
  • Truncate Chars (6): {{ line|truncatechars:6 }}

Number filters for the number {{number}}

  • Floatformat (2): {{ number|floatformat:2 }}
  • Add 100: {{ number|add:100 }}
  • Divisible by 3: {{ number|divisibleby:3 }}

List filters for the list {{list}}

  • Length: {{ list|length }}
  • First Element: {{ list|first }}
  • Last Element: {{ list|last }}
  • Join Elements: {{ list|join:", " }}
  • Slice (first 2): {{ list|slice:":2" }}

Date and Time filters

  • Current Date & Time: {{ my_date }}
  • Formatted Date: {{ my_date|date:"d-m-y" }}
  • Time left until my birthday: {{ fut_date|timeuntil }}

Escape Filters

{% comment %} means closing execution of any code like JS, HTML etc {% endcomment %} {% comment %} Shows an alert with the message "Hello, World!" if escapejs is not applied {% endcomment %}
{{ my_html|safe }}
{{ my_html|force_escape }}

Conditional Variable: {{ cond_var }}

  • Conditional Variable with default: {{ cond_var|default:"No value provided" }}
  • Conditional Variable with default_if_none: {{ cond_var|default_if_none:"Not Found" }}
  • Conditional Variable Yes/No: {{ cond_dec|yesno:"Decision Taken, Decision not Taken" }}

    • Dictionary List Filters

    • Dictionary: {{ dict_list}}

    • Sorted with age in reverse: {{ dict_list|dictsortreversed:'age'}}
